# 🏠 House Price Prediction App

An AI-powered web application that predicts house prices based on user inputs using Machine Learning.

---

## 🚀 Live Demo

🔗 https://houseprediction-dn5iutwpnaqwacjffg5uxf.streamlit.app/

---

## 🧠 Features

* 📊 Predict house prices instantly
* 📂 Bulk prediction using CSV upload
* 🧠 Explainable AI using SHAP
* 📈 Data visualization
* 🎯 Accuracy score (R²)
* 💡 Price category (Affordable / Mid-range / Premium)

---

## ⚙️ Tech Stack

* Python
* Streamlit
* Scikit-learn
* SHAP
* Pandas / NumPy

---

## 📊 Model Details

* Model: HistGradientBoostingRegressor
* Features:

  * Size
  * BHK
  * Rooms
  * Location
  * Area Type

---

## 📂 How to Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO
pip install -r requirements.txt
streamlit run app.py
```

---

## 📁 CSV Format for Bulk Prediction

```csv
size,bhk,rooms,location,area_type
1200,2,2,Chennai,Super built-up Area
1500,3,3,Bangalore,Built-up Area
```
